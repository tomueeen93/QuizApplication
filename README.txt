これは法政大学情報科学部の2014年度秋学期授業「自然言語とコンピューティング」において作られたアプリケーションです。

作成者:	12k0102 植草 智輝
Email:	tomoki.uekusa.8x@stu.hosei.ac.jp

実行環境
VisualStudio2013 for WEB上で動くプログラムになっています。

使い方
VisualStudio2013 for WEB以上のバージョンで中にある.slnファイルを開いて下さい
デバッグとしてGoogleChromeで実行すると動きます。

注意事項
各プログラムの場所については基本的にASP.NETのフレームワークに準拠しています。
レイアウトについてはGoogleChrome以外で実行すると崩れてしまいます。
実行しても問題データが読み込まれない場合がありますが、何度かデバックし直すか、キャッシュをクリアすると直ると思います。
